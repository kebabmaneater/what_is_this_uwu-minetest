# What is this? UwU

A mod that says at the top of your screen what you're looking at.

## Screenshots:

![](./prints/dirt_with_grass.png)

![](./prints/grass.png)

![](./prints/water_source.png)

## Chatcommands

To unshow the pop-up, type this in console:

```
/wituwu
```

Retype to show it again.
